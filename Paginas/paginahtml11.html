Creación de página web en HTML.

Estos son los pasos para crear una página web:
Primero, se definen los requerimientos básicos para usarse como una guía o un plano del proyecto.
Segundo, se crea un prototipo sin muchos detalles usando como guía la lista de requerimientos, solo para visualizar la estructura final.
Tercero, se comienza a escribir el código usando el prototipo como guía.

Requerimientos básicos.

Escribir los requerimientos y la estructura del sitio web te permite visualizar lo realmente necesario y ver cosas que faltan, sobran, cambiar, reemplazar, modificar y todo sin necesidad de hacer código.
  1. Inicio (home).
    1. Logo y menú (header).
    2. Banner.
      1. Título.
      2. Descripción.
      3. Botón.
      4. Imagen.
    3. Servicios que ofrecemos.
      1. Desarrollo web.
      2. Desarrollo móvil.
      3. Sistemas a medida.
    4. Banner corporativo.
      1. Imagen de fondo.
      2. Texto principal.
      3. Texto secundario.
      4. Botón que lleve a contacto.
    5. Últimos proyectos.
      1. Mostrar algunos proyectos.
        1. Imagen (Screenshots).
        2. Título (nombre del cliente).
        3. Breve descripción.
    6. Footer.
      1. ¿Quieres trabajar con nosotros?
        1. Texto adicional
        2. Botón de contacto.
      2. Redes sociales.
      3. Copyright.
  2. ¿Quiénes somos?
    1. Sobre nosotros.
      1. Imagen.
      2. Descripción.
    2. Misión.
      1. Imagen.
      2. Descripción.
    3. Visión.
      1. Imagen.
      2. Descripción.
  3. Proyectos.
    1. Cada proyecto:
      1. Imagen (Screenshot).
      2. Nombre del cliente o proyecto.
      3. Descripción.
      4. Link al proyecto
  4. Contactos.
    1. Datos de la empresa.
      1. Teléfono.
      2. Dirección.
      3. Correo.
      4. Mapa de ubicación.
    2. Formulario de contacto.
      1. Integración con algún servicio (Puede ser FB o Messenger).

Visualización.

Visualizar el sitio web es como realizar un mapa o una maqueta de un posible resultado.
Una vez realizado el wireframe se debe mostrar al cliente para que pueda observar y dar el visto bueno, suele ocurrir que el cliente pida modificaciones y se debe de aplicar los cambios sugeridos.

Código.

El wireframe (Home) será el guía al iniciar a escribir el código del sitio web.
Es buena práctica colocarle al código comentarios sobre cada bloque para ayudar a guiar futuras modificaciones en él.

Banner.

body
main
section
div
h1

img
Con la etiqueta main se coloca el contenido principal de la página.
La sección Banner será la primera sección principal que se indicará con la etiqueta section, dentro de esta se colocará un contenedor con la etiqueta div y dentro de este contenedor del Banner se colocarán dos elementos que se presentan como dos sub-contenedores, nuevamente en código se escribirán con div. El primer sub-contenedor contiene el título principal h1 y el texto de presentación que se colocará con la etiqueta p. En el segundo sub-contenedor tiene solo la imagen, la cual se colocará con la etiqueta div y dentro de ella la etiqueta img.

Todavía dentro del main se crea otra sección con la etiqueta section, se colocará un subtítulo con la etiqueta h2 y además se coloca su contenedor con una etiqueta div, dentro del contenedor div existen tres componentes para los cuales se colocará la etiqueta article. En cada article se tiene la misma estructura, un ícono que se delimita con la etiqueta span, un sub-subtítulo h3 y un texto o párrafo con la etiqueta p.

Todavía dentro del main se crea otra sección con la etiqueta section, se coloca su contenedor con una etiqueta div, dentro del contenedor div se coloca un subtítulo h2, un texto de descripción que se colocará con la etiqueta p y el botón de contacto que se creará usando la etiqueta div y dentro de ella la etiqueta a.
La imagen de fondo se coloca desde el archivo de estilos CSS.

Dentro de cada enlace a se coloca un artículo usando la etiqueta article, éste a su vez contendrá un contenedor que se define con la etiqueta div, dentro de éste contenedor se le colocará un sub-subtítulo con la etiqueta h4 y un texto de descripción que se colocará con la etiqueta p.

Se cierra el main.
