Listas

Ordenadas:

Se crean con las etiquetas ol (de ordered) con su inicialización y finalización, dentro de esta etiqueta se coloca la etiqueta li (de list item)
con su inicialización y finalización para colocar cada uno de los ítems. Estas listas tienen una numeración.
Dentro de cada etiqueta ol se pueden colocar otras ol para crear sub listas.
  1. Ítem
  2. Ítem
  1. Ítem
    1. Sub-ítem
      1. Sub-sub-ítem
  2. Ítem
Se les pueden agregar atributos a las listas como:
<ol reversed> este atributo invierte el orden numérico de las listas ordenadas.
<ol start="#"> este atributo hace que la lista ordenada comience en el número que se le asigna.
<li value="#"> este atributo del ítem hijo (de la lista ordenada) hace que el ítem empiece desde el número especificado y todos los ítems que le preceden continúan con la numeración de esta alteración.

Desordenadas:

Se crean con las etiquetas ul (de unordered list) con su inicialización y finalización, dentro de esta etiqueta se crea la etiqueta li (de list item) con su inicialización y finalización para colocar cada uno de los ítems.
Dentro de cada etiqueta ul se pueden colocar otras ul para crear sub listas.

  • Ítem 1
  • Ítem 2
  • Ítem 1
    • Sub-ítem
      • Sub-sub-ítem
        • Sub-sub-sub-ítem
  • Ítem 2


Si no se especifica el tipo de lista con ol o ul se da por default la lista desordenada ul pero no es recomendable no especificar el tipo de lista.

Anidadas:

Las listas anidadas son las combinadas ol y ul.

    Listas de definición

    Se crean con las etiquetas dl (de definition list) con su inicialización y finalización, dentro de esta etiqueta se coloca la etiqueta dt (de definition term) con su inicialización y finalización que es donde se colocan los términos, afuera de esta etiqueta se coloca la etiqueta dd (de definition description) con su inicialización y finalización que es donde se colocan las definiciones para los términos.
