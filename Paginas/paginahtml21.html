Accesibilidad Web.

La accesibilidad web es la práctica continua de asegurarnos que todo lo que creamos para la web se puede usar, interpretar y operar por una variedad de personas en una variedad de situaciones.

La accesibilidad web es importante para incluir a personas en situaciones de discapacidad, para mejorar la usabilidad de nuestros proyectos y porque en algunos paises la ley pide incluír algunas características en las páginas web.

Abreviaciones importantes.

a11y

A11y es un “numerónimo” (una palabra basada en números) para la palabra “accessibility” (“accesibilidad”). El 11 representa las letras entre la “a” y la “y“, similar a i18n para internacionalización. Esta abreviatura también se utiliza como una forma de activismo para impulsar un Internet más accesible y amigable para las personas con discapacidad por lo que muchos articulos y documentación relacionada con la accesibilidad se ligan a este numerónimo.

WAI

Web Accessibility Initiative es una iniciativa dentro de W3C cuya meta es asegurar la accesibilidad web.

WCAG

En inglés son las Web Content Accessibility Guidelines o en español las Pautas de Accesibilidad para el Contenido Web. Un rango de recomendaciones para crear contenido web más accesible.

Principios de conformidad.

Existen 3 niveles de conformidad para medir la accesibilidad en un producto:

  • A - básica
  • AA - intermedia
  • AAA - alta

Estos niveles se aplican a cada uno de los criterios de conformidad los cuales se encuentran dentro de los 4 principios:

  1. Perceptible.

    2. La información y los componentes de la interfaz de usuario deben ser presentados a los Usuarios de modo que ellos puedan percibirlos.

  2. Operable.

    3. Los componentes de la interfaz de usuario y la navegación deben ser operables.

  3. Comprensible.

    4. La información y el manejo de la interfaz de usuario deben ser comprensibles.

  4. Robusto.

    5. El contenido debe ser suficientemente robusto como para ser interpretado de forma fiable por una amplia variedad de aplicaciones de usuario, incluyendo las ayudas técnicas.

Tecnologías asistivas.

Son alternativas a las existentes para consumir contenido:

  • Visuales
    • Lectores de pantallas.
    • Extensiones que manipulan el CSS: son modifica el CSS para el mejor consumo de contenido para el usuario como agrandar el texto o modificar los colores.
  • Motoras
    • Varilla bucal.
    • Switch: pueden ser configurables, sensibles, activables por movimientos de cabeza o gestos, simples, mecánicos, etc.
