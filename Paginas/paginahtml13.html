Servidores

Subir proyectos a la web.

Netlyfy

Netlify es una plataforma de desarrollo web que permite subir aplicaciones de cualquier tipo a la nube.
Entre sus servicios cuenta con Netfly Drop en el cual, arrastrando una carpeta (con un proyecto de página web) y soltándola en el área que se indica en la página, nos entrega un link, con él podremos visualizar nuestro proyecto en algún navegador web.
Al momento en que entregue el link del proyecto subido se puede comprar un dominio personalizado, después de realizar la transacción de compra se debe esperar entre media hora y tres horas para que los DNS se propaguen y así el sitio web funcione.

Comprar dominios.

Los dominios son en donde se alojan los proyectos que se crean, es la ruta que busca un usuario en un navegador para ver el proyecto o aplicación creada como https://dominio.com

Hostinger

Es un proveedor de alojamiento web y dominios web.

El proveedor ofrece de forma gratuita la SSL (seguridad en la capa de transporte). Además, el proveedor ofrece otras herramientas por las que se paga un extra como un gestor de correo o un creador de páginas para facilitar la creación y diseño de la página, entre otras.

Open Graph

Son una serie de etiquetas que se añaden al código de la página, invisibles de cara al visitante de la web, que ayudan a Facebook y a otras redes sociales como Twitter y Google+ a identificar los elementos principales del artículo que se quiere compartir: título, imagen, descripción, etc.
