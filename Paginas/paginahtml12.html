Creación de página web en CSS.

:root

Es recomendable colocar nuestros estilos repetitivos en nuestra clase :root porque esta raíz es accesible a todo nuestro código y si se desea modificar un estilo, se puede modificar desde el :root en vez de buscar la línea especifica en donde se encuentra el estilo.
Primero se declaran las variables para el Mobile first, o sea, para pantallas de tamaño celular, después de finalizar el :root se coloca un media query para colocar los valores en pantallas de tamaño computador.
Primero, se declaran los colores, si se decide cambiar un color, se puede camiar desde la raíz y evitar buscar la línea en la que se aplica el color dentro del código.
Segundo, se declaran las variables que contendrán las fuentes tipográgicas, si se usan fuentes externas, se deben importarantes del :root.
Tercero, se declaran en variables los tamaños como ancho máximo (max-width), espaciado (margin-bottom) o alto de líneas (line-height).
Cuarto, se declaran los tamaños de las fuentes de los titulos y contenidos.
Quinto, se declara el apilamiento de capas, el orden en que aparecen las etiuetas unas atras de otras.
Sexto, se declaran otras variables aparte como el redondeado de los bordes o la sombra.
