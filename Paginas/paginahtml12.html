Prototipado.

Se crea un prototipo sin muchos detalles usando como guía la lista de requerimientos, solo para visualizar la estructura final. Después, ya cuando sea aprobado el prototipo y no haya grandes modificaciones, se detalla el proyecto con colores o medidas específicas.

Figma es una gran herramienta para prototipar.

Visualización.

Visualizar el sitio web es como realizar un mapa o una maqueta de un posible resultado.
Una vez realizado el wireframe se debe mostrar al cliente para que pueda observar y dar el visto bueno, suele ocurrir que el cliente pida modificaciones y se debe de aplicar los cambios sugeridos.
