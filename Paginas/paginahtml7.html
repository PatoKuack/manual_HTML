Tablas

Las tablas sirven para tabular contenido es decir llenarlas de datos, mostrar valores, etc.
La etiqueta table con su inicialización y finalización se utiliza para crear tablas de contenidos, similar a la estructura de las hojas de cálculo.
La etiqueta tr con su inicialización y finalización se utiliza para crear las filas.
La etiqueta th con su inicialización y finalización se utiliza para crear la cabecera de la tabla, es decir las filas principales de cada columna, se coloca dentro de los "tr".
La etiqueta td con su inicialización y finalización se utiliza para crear las columnas, se coloca dentro de los "tr".
Con el atributo border=1 dentro de la etiqueta "table" indica que la tabla tendrá un borde de 1cm de ancho, aunque no es recomendable usarlo porque el diseño debe de hacerlo CSS.
La etiqueta thead con su inicialización y finalización se utiliza para marcar la semántica de la tabla (entidades), los "tr" y "th" van dentro de estas etiquetas.
La etiqueta tbody con su inicialización y finalización se utiliza para marcar la semántica de la tabla (atributos de las entidades), los "tr" y "td" van dentro de estas etiquetas.
La etiqueta tfoot con su inicialización y finalización se utiliza para marcar la semántica de la tabla (pie de tabla), los "tr" y "td" van dentro de estas etiquetas. Aquí se colocan las conclusiones de las columnas, como un total de facturación de una cuenta. Siempre se auto coloca al final de las filas.
La etiqueta caption con su inicialización y finalización se utiliza para colocar el título de la tabla (va dentro de la etiqueta "table"). Siempre se auto coloca arriba de la tabla, aunque la coloquemos hasta el final.

Agrupación de filas y columnas

El atributo colspan=2 se utiliza para que una celda ocupe determinados espacios de columnas, en este caso una celada sería del tamaño de dos celdas en lo largo (ejemplo: td colspan=2>).
El atributo rowspan=2 se utiliza para que una celda ocupe determinados espacios de filas, en este caso una celada sería del tamaño de dos celdas en lo alto (ejemplo: td rowspan=2>).
(Si no crean su función "rowspan" y "colspan" se deberán crear más filas o columnas)

Para colorear toda una fila, solo se una un "backgroun" o "class" en las "tr" pero para colorear solo una columna...
La etiqueta colgroup con su inicialización y finalización permite seleccionar grupos de columnas completas.
Dentro de esa etiqueta se utiliza la etiqueta col solo con su inicialización se utiliza para indicar cada columna (si se tiene una tabla de 3 columnas se tendrán q poner 3 etiquetas col).
Se puede también colocar únicamente un "col" indicando con un atributo "span=N" que la agrupación se hará para la ENECIMA columna, si se hace esto se usará otro "col" para indicar con un atributo "style=background:" o "class" el color de la columna.
Con CSS se puede utilizar esta etiqueta para dar formato a determinado grupo de celdas.
El atributo class="col-1" se utiliza para CSS indicando que esa columna se podrá mandar a llamar como "col-1" (se tendría q colocar un "col" por columna ej. col class="col-1">).
