Metadatos

Es información que le damos al navegador web o a los buscadores para que identifiquen información. Nos sirven para darle una descripción de nuestra página web al navegador y nuestra página la puedan posicionar dependiendo de lo que busquen los usuarios.
Se usan con la etiqueta meta, no necesita finalización y se coloca en la sección head.
Hay varios metas como:

Description.

Con el atributo name se define el tipo de meta que vamos a utilizar que en este caso sería el meta "description".
En el atributo content colocamos la descripción de nuestro sitio web, ésta se muestra en el navegador abajo del título (se recomienda que el fragmento de texto no rebase los 150 o 170 caracteres).
(ejemplo: meta name="description" content="descripción de la página"/>).

Keywords.

Con el atributo name se define el tipo de meta que vamos a utilizar que en este caso sería el meta "keywords".
En el atributo content colocamos palabras claves de nuestro sitio web separadas por comas (estos metadatos ya no los toma en cuenta Google para el buscador, sin embargo, otros buscadores si los toman en cuenta por lo que no está de más agregarlos).
(ejemplo: meta name="keywords" content="curso, básico, html"/>).

charset

La etiqueta meta charset="UFT-8" se coloca dentro de head y es un codificador de caracteres unicode para que la página soporte simbología especial como tildes.

Enlazar archivos CSS o JS.

Para enlazar una hoja de estilo CSS se usa la etiqueta link y no necesita etiqueta de finalización (link rel= ... .css>), va en la sección head.
La etiqueta link permite traer recursos externos a la página web.
Los atributos de ésta etiqueta son:
rel="icon" href="ruta de la imagen" indican al navegador que coloque en el icono de la página la imagen de la ruta especificada.
rel="stylesheet" href="ruta del documento CSS" indica que se aplicará el estilo CSS de la ruta especificada.
href="estilos/estilo1.css" indica la dirección de la hoja de estilos, se puede poner una dirección absoluta (https//:www...) o una relativa.
(no recomendado) Si se desea crear un CSS rápido se puede usar la etiqueta style con su inicialización y finalización en la sección head, adentro de estas van códigos como background: green; o color: black;
A esta etiqueta se le agregaba antes el atributo type="text/css" para indicar su tipo de contenido, pero ahora ya no es necesario sin embargo se pueden encontrar en códigos viejos.

Para enlazar una hoja de tipo java script se usa la etiqueta src="javascript.js" al final de la sección head para que se cargue todo el diseño y después la interacción de la página.
Si se desea crear un CSS rápido se puede usar la etiqueta script con su inicialización y finalización al final de la sección body, dentro de esta etiqueta va escrito el código javascript.
A esta etiqueta se le agregaba antes el atributo type="text/javascript" para indicar su tipo de contenido, pero ahora ya no es necesario sin embargo se pueden encontrar en códigos viejos.

Cache.

Cuando se carga una página en el navegador de un usuario, la página se guarda en la memoria cache evitando que el navegador cargue futuras actualizaciones de la página, por esto se utilizan los siguientes metadatos:
<meta http-equiv="cache-control" content="no-cache"> <meta http-equiv="Pragma" content="no-cache"> <meta http-equiv="expires" content="0">
