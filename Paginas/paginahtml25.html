Navegación con teclado.

El teclado es muy importante en la accesibilidad web porque las tecnologías asistivas tienden a conectarse con el teclado para darle a sus usuarios diferentes maneras de navegar cuando no tienen acceso a un mouse o un teclado.

Los elementos que reciben foco de teclado son elementos interactivos:

  • Enlaces: redireccionan a otra página, son para la navegación.
  • Botones: ayudan a interactuar con la página de alguna manera.
  • Formularios: requieren una interacción del usuario para llenar los datos.

Outline

Los estilos de outline de los elementos son importantes para que las personas que utilizan el teclado para navegar en las páginas web vean en donde estan posicionadas, generalmente se utilizan las pseudoclases :focus, :focus-widthin, :focus-visible para modificar los estilos de los elementos interactivos y sea notable que se les esta haciendo focus.

Tabindex

Indica si su elemento puede ser enfocado, y si participa en la navegación secuencial del teclado.

Recuerda que los elementos enfocables son enlaces, botones y elementos de un formulario y son estos a los que se les coloca el atributo tabindex.

Valores de Tabindex:

  • Valor negativo(-1): El elemento debe ser enfocado, pero no debe de ser accesible a través de la navegación con teclado.
  • Valor positivo (>0): Debe poder ser enfocado y su orden relativo es definido por el valor del atributo. Se puede definir un orden secuencial del teclado colocando este valor a los elementos (1 primero en enfocarse, ..., +n último en enfocarse), pero se debe tener cuidado con su uso porque una mala gerarquia de valores puede romper el sitio web.
  • Valor de 0: Debe ser enfocado y ser accesible a través de la navegación secuencial del teclado, pero su orden relativo es definido por convención de la plataforma, o sea, permite seguir el orden normal de la estructura HTML.

Cuando se aplica tabindex a interacciones, se suele utilizar con JavaScript con el método element.setAttribute("tabindex", "-1"); para colocar el atributo a una etiqueta o con el método element.removeAttribute("tabindex"); para remover el atributo de la etiqueta.

Keycode.info

Esta página puede escuchar cualquier tecla del teclado y mostrar la clave de la tecla, por ejemplo, si se presiona la tecla Esc se otendrá el numero 27 que es la clave de esa tecla y se puede aplicar de la siguiente forma:

document.body.addEventListener("keyup", e => listenkey(e));
function listenkey(e) {
 if (e.keycode === 27){...}
}
